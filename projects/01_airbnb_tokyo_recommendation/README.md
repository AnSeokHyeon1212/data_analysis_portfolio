# 도쿄 Airbnb 가성비·가심비 숙소 추천 분석

## 1. 프로젝트 한 줄 소개

도쿄 Airbnb 데이터를 기반으로 가격, 위치, 평점, 리뷰 수를 종합해 사용자 관점의 숙소 추천 기준을 설계한 프로젝트입니다.

## 2. 프로젝트 배경

여행 숙소 선택 과정에서 사용자는 가격뿐 아니라 위치, 평점, 리뷰 수, 숙소 조건을 함께 고려합니다.  
본 프로젝트는 사용자의 숙소 선택 의사결정을 돕기 위해 데이터 기반 추천 기준을 설계했습니다.

## 3. 문제 정의

- 단순 최저가 숙소 추천은 사용자 만족도를 보장하기 어려움
- 가격, 위치, 평점, 리뷰 신뢰도를 함께 고려한 기준 필요
- 사용자가 비교하기 쉬운 추천 기준 설계 필요

## 4. 데이터 설명

- 데이터: 도쿄 Airbnb 숙소 데이터
- 주요 변수: 가격, 위치, 평점, 리뷰 수, 숙소 유형, 수용 인원
- 분석 단위: 개별 숙소

## 5. 분석 목표

- 숙소 선택에 영향을 주는 주요 요인 파악
- 가격 대비 만족도가 높은 숙소 조건 도출
- 사용자 관점의 추천 기준 설계

## 6. 사용 기술

- Python
- pandas
- numpy
- matplotlib
- seaborn

## 7. 분석 프로세스

1. 데이터 수집 및 구조 확인
2. 결측치 및 이상치 처리
3. 가격, 평점, 리뷰 수 분포 분석
4. 지역별·숙소 유형별 차이 분석
5. 가성비·가심비 추천 기준 설계
6. 결과 시각화 및 인사이트 도출

## 8. 주요 결과

- 가격만으로는 숙소 만족도를 설명하기 어려움
- 평점과 리뷰 수를 함께 고려할 때 추천 신뢰도 향상 가능
- 사용자 입장에서 비교 가능한 추천 기준 제안
- 2024 교내 경영빅데이터 분석 경진대회 준우수상 수상

## 9. 인사이트

숙소 추천은 단순 가격 비교가 아니라 사용자의 선택 기준을 데이터로 구조화하는 문제입니다.

## 10. 한계점 및 개선 방향

- 실제 예약 전환 데이터가 없어 추천 효과 검증에는 한계
- 관광지, 지하철역, 상권 데이터 결합 필요
- 향후 사용자 유형별 추천 기준으로 확장 가능

## 11. 내가 맡은 역할

- 데이터 전처리
- 탐색적 데이터 분석
- 추천 기준 설계
- 발표 자료 구성

## 12. 직무 연결 포인트

서비스/프로덕트 데이터 분석 직무에서 필요한 사용자 선택 기준 정의, 지표 설계, 분석 결과 해석 경험으로 연결됩니다.
